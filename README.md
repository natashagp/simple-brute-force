# Simple Brute Force

### Step 1

Clone the repository

```shell
git clone https://github.com/natashagp/simple-brute-force.git
```

### Step 2

Change the directory to simple-brute-force

```shell
cd simple-brute-force
```

### Step 3

Run project

```shell
python main.py
```
